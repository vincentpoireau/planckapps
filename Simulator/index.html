Simulateur du fond diffus cosmologique

Ingrédients de l'Univers

Matière normale - Tout ce que vous voyez autour de vous : les tables, les bâtiments, les planètes, les étoiles, etc.

Matière noire - Cette matière n'est pas visible, et n'a jamais été détectée en laboratoire. Son influence gravitationnelle est observée, entre autres, dans la rotation des galaxies. Elle est appelée noire (ou sombre) parce qu'elle ne produit pas de lumière.

Énergie noire - L'énergie noire excerce une pression qui accélère l'expansion de l'Univers. Elle est appelée noire (ou sombre) parce que l'on ignore ce qu'elle est réellement.

À propos de Planck

Le satellite Planck (http://www.esa.int/planck) est une mission de l'agence spatiale européenne (ESA) lancée en mai 2009. Il s'agit de l'un des satellites les plus avancés technologiquement jamais lancés. Planck a observé le cosmos pour chercher l'origine de notre Univers. Ses premiers résultats ont été rendus publics en mars 2013.

Crédits

Le code du simulateur dans sa version anglaise est disponible sur GitHub (https://github.com/chrisnorth/planckapps/tree/master/Simulator).
